# E-Commerce SQL Analysis

HarryShop eCommerce Data Analysis (SQL)

📌 Project Overview
Ye project ek eCommerce platform (**HarryShop**) ka database design aur data analysis hai. Ismein customer management, product inventory, order tracking, aur payment processing ko SQL ke zariye handle kiya gaya hai.

## 🗄️ Database Schema
Database mein total 5 tables hain:
1. **Customers**: User profile aur location data.
2. **Products**: Merchandise categories (Clothing, Accessories, Stationery).
3. **Orders**: Order status (Delivered, Pending, Cancelled) track karne ke liye.
4. **Order_Items**: Har order ki quantity aur product mapping.
5. **Payments**: Payment modes (UPI, Credit Card, Debit Card) aur amounts.

## 🚀 Key Business Insights
Maine SQL queries use karke ye major insights nikale hain:
- **Total Revenue**: Poore shop ka total sales collection.
- **Top Selling Products**: Sabse zyada bikne wale items (jaise Python Hoodies, etc.).
- **High-Value Customers**: Wo log jo shop par sabse zyada spend kar rahe hain.
- **Order Status Tracking**: Cancelled aur pending orders ka count.

## 🛠️ Tech Stack
- **Database**: MySQL
- **Language**: SQL
